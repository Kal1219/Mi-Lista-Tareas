# 📌 Lista de Tareas con React y LocalStorage

Este proyecto es una **lista de tareas** (To-Do List) creada con **React** que permite:

✅ Agregar tareas 🆕  
✅ Marcar tareas como completadas ✅  
✅ Eliminar tareas ❌  
✅ Editar tareas ✏️  
✅ Guardar tareas en `localStorage` 📂 (persistencia de datos)  

---

## 🚀 **Tecnologías utilizadas**

- [React](https://react.dev/) ⚛️
- [Vite](https://vitejs.dev/) ⚡
- JavaScript (ES6+)

## 🛠 **Instalación y Ejecución**

### 1️⃣ Clonar el repositorio
```bash
git clone https://github.com/tu-usuario/mi-lista-tareas.git
cd mi-lista-tareas
```

### 2️⃣ Instalar dependencias
```bash
npm install
```

### 3️⃣ Ejecutar el proyecto
```bash
npm run dev
```
Luego, abre [http://localhost:5173](http://localhost:5173) en tu navegador. 🚀

---

## ✨ **Funciones principales**

🔹 **Agregar tareas**: Escribe una tarea y presiona el botón "Agregar".  
🔹 **Marcar como completada**: Haz clic en ✅ para tachar una tarea.  
🔹 **Eliminar tareas**: Presiona ❌ para borrar una tarea.  
🔹 **Editar tareas**: Presiona ✏️ para modificar el texto de la tarea.  
🔹 **Persistencia**: Las tareas se guardan en `localStorage`.
